# Requirement

* RxSwift 5.0.1
* RxCocoa 5.0.1

# Installation

このアプリは`Cocoa Pods`を使って、ライブラリを管理しています。

下記コマンドでビルドする前にライブラリをインストールしてください。

```bash
pod install
```
